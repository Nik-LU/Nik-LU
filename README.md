## Android developer

Пет-проекты для демонстрации архитектуры, чистого кода и подходов.

### Projects
В профиле закреплены 4 проекта (каждый — отдельный репозиторий с описанием решений и запуском).

## Стек
- **Языки:** Kotlin / Java
- **UI:** Jetpack Compose + Compose Canvas / XML
- **Screens:** Activity / Fragment / Compose
- **Navigation:** Intents  / Jetpack Navigation  / Decompose
- **Архитектура:** MVVM / MVI (UDF), Clean Architecture (data/domain/presentation)
- **Android:** Service / WorkManager / BroadcastReceiver / ContentProvider 
- **Async:** Coroutines, Flow, LiveData
- **DI:** Hilt / Dagger2
- **Network:** Retrofit/OkHttp
- **Data:** Room, DataStore
- **Storage:** Internal storage
- **Прочее:** Firebase
